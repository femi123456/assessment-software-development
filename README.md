## Overview
The Software Development Unit (SDU) is a structured learning community focused on building practical software development skills using modern tools, prompt engineering, and ethical development practices. The program is divided into three progressive tracks: Beginner, Intermediate, and Advanced.


---

## Learning Tracks

### Beginner Track
Designed for members who are new to software development and prompt engineering.

#### Requirements
- Basic understanding of computers and the internet
- Interest in learning Prompt Engineering
- GitHub account
- GitHub Student Developer Pack
- Access to Prompt Engineering learning materials

#### Projects
Participants will build beginner-friendly projects such as:
- E-commerce website (basic functionality)
- Personal portfolio website
- To-Do list application
- Blog website
- Simple landing page

#### Work Ethics
Members are expected to follow:
- Proper documentation
- Clean and readable code
- Consistent and meaningful GitHub commits
- Collaboration and respect for deadlines

---

### Intermediate Track
For members with prior development experience who want to integrate prompt engineering into real development workflows.

#### Requirements
- Understanding of Prompt Engineering concepts
- GitHub account
- GitHub Student Developer Pack
- Access to Prompt Engineering learning materials
- Strong knowledge of either Frontend or Backend development (preferably both)

#### Projects
- Build a project using your preferred tech stack, such as:
  - JavaScript
  - Python
  - Node.js
  - Django
- Use **Lovable/v0.dev/** in conjunction with your chosen stack to enhance productivity

#### Work Ethics
Participants must demonstrate:
- Proper documentation
- Modular and maintainable code
- Use of GitHub issues, branches, and pull requests
- Ethical and responsible use of AI tools

---

### Advanced Track
For experienced developers with real-world software deployment experience.

#### Requirements
- Present an application you have built that:
  - Is used by real users
  - Is deployed in a production environment
  - Demonstrates real-world impact (paid or experience-based)

#### Evaluation Criteria
- Problem-solving approach and impact
- Code quality and architecture
- Scalability and performance

#### Work Ethics
Advanced members are expected to uphold:
- Detailed and structured documentation
- Secure and scalable development practices
- Performance optimization and testing
- Mentorship and contribution to the SDU community

---

## Progression
Members may progress through tracks based on skill growth, project quality, and demonstrated experience.

---

## Contribution & Conduct
All members are expected to maintain professionalism, respect, and ethical development practices throughout their participation in SDU.
